CONTENIDO TÉCNICO RESUMIDO
================================================================================

MÓDULOS IMPLEMENTADOS:
├─ Inventario: Entrada/salida de materia prima con trazabilidad
├─ Producción: Órdenes de producción con consumo real y cálculo de merma
├─ Ventas: Registro de ventas con margen de ganancia
├─ Nómina: Cálculo automático de salarios con aportes y descuentos
├─ Auditoría: Trazabilidad completa de operaciones
└─ Reportes: PDF y Excel para análisis

PRODUCTOS ESPECIFICADOS:
├─ Papas fritas: 45gr, 90gr, 190gr, 500gr, 850gr, 1kg
├─ Palitos: 100gr, 250gr, 500gr, 1kg
├─ Maní: 90gr, 190gr, 500gr, 1kg
├─ Chizitos/Puflitos: 150gr, 300gr, 1kg
├─ Mix cervecero: 90gr, 190gr, 500gr
└─ Panificados: Prepizza, panes de hamburguesa, de Viena, etc.

MATERIA PRIMA:
├─ Papa, aceite, sal, envase, harina, grasa, azúcar
├─ Levadura, malta, antimoho, aditivos, salsa, vinagre, alcohol
├─ Maní, y gestión de costos: luz, gas, agua, impuestos

CARACTERÍSTICA DESTACADA - NÓMINA:
├─ Cálculo automático de:
│  ├─ Sueldo base + antigüedad (años × porcentaje)
│  ├─ Aportes AFIP configurable (ej: 17%)
│  ├─ Descuentos sindicales/otros
│  ├─ Neto a pagar
│  └─ Costo total empresa (incluyendo aportes)
├─ Procesamiento mensual automático
└─ Recibos en PDF individuales

TECNOLOGÍA RECOMENDADA
================================================================================

BACKEND:
├─ Node.js + Express.js O Python + FastAPI
├─ MySQL 8.0+ / PostgreSQL 12+
├─ Redis para caching
└─ JWT para autenticación

FRONTEND:
├─ Angular 20+ con TypeScript
├─ Material UI / Tailwind CSS
└─ Responsive (desktop + mobile)

DEVOPS:
├─ Docker para containerización
├─ GitHub Actions / GitLab CI para CI/CD
├─ 2-3 instancias con load balancer en producción
└─ Monitoreo con New Relic / DataDog

================================================================================
BENEFICIOS ESPERADOS
================================================================================

OPERACIONALES:
├─ Reducción 80% en tiempo de nómina (2 días → 30 min)
├─ Eliminación de errores manuales en cálculos salariales
├─ Visibilidad en tiempo real del inventario
├─ Alertas automáticas de stock bajo
└─ Cálculo automático de costos de producción

FINANCIEROS:
├─ Control preciso de márgenes por producto
├─ Identificación de ineficiencias (merma)
├─ Mejor control de cash flow
└─ Análisis de rentabilidad por línea

REGULATORIOS:
├─ Compliance con regulaciones laborales (AFIP)
├─ Auditoría completa para organismos reguladores
├─ Datos listos para auditoría externa
└─ Reportes para presentación a organismos

ESTRATÉGICOS:
├─ Base de datos centralizada
├─ Decisiones basadas en datos
└─ Escalabilidad para crecimiento

================================================================================
SEGURIDAD IMPLEMENTADA
================================================================================

├─ Autenticación JWT con expiración automática
├─ Autorización basada en 5 roles:
│  ├─ ADMIN (acceso total)
│  ├─ GERENTE (producción, ventas, reportes)
│  ├─ OPERARIO (inventario, producción)
│  ├─ RRHH (empleados, nómina)
│  └─ CONTADOR (auditoría, reportes)
├─ Contraseñas hasheadas con bcrypt
├─ HTTPS obligatorio
├─ Rate limiting (100 req/min por usuario)
├─ Auditoría de todas las operaciones
├─ Encriptación de datos sensibles
└─ Validación de entrada (SQL injection, XSS)
